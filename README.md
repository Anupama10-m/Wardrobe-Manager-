*Smart Wardrobe Matcher*

Smart Wardrobe Matcher is a frontend web project made to suggest outfit combinations from clothes added by the user.

The idea came from a simple problem — sometimes we have clothes but do not know what combinations look good together. This project lets users add clothing items and generates outfit suggestions based on matching clothing styles.

Example:

Formal Shirt + Formal Pants → Suggested
Formal Shirt + Casual Pyjama → Not Suggested

Users can also save outfit combinations they like.

 *Features*

* Add clothing items
* Choose clothing type (Top / Bottom)
* Choose clothing style
* Suggest matching outfits
* Save outfit combinations
* Delete clothing items
* Responsive dark theme UI

 *Tech Stack*

* HTML
* CSS
* JavaScript

 *How It Works* *

1. Enter a clothing item
2. Select whether it is a Top or Bottom
3. Choose its style
4. Add it to the wardrobe
5. Click Suggest Outfit
6. View and save outfit combinations

The matching logic currently suggests outfits only when both clothing items belong to the same style category.
